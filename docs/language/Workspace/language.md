# Workspace

The workspace is what the author inhabits. Each sigil opens in its own native window. Inside, the author navigates the sigil structure, writes @Language, anchors purpose through a @Vision, thinks with a @DesignPartner, and flattens the whole structure for handoff via @Export.

In the tree panel, arrow keys navigate between sigils. Enter creates a new sibling at the same level. Keyboard shortcuts are configurable in Settings.
