# Attention

A sigil is a named pattern that attention recognizes. Its boundary makes it a bounded context. Inside that boundary, it can define up to five contained sigils. A sigil is always contained within another sigil. What appears as a root is only the sigil currently opened.

A sigil without attention is inert. When human or AI attention attends to a sigil, it comes alive. Multiple attentions can inhabit the same sigil at once.

Inside a sigil, domain language is written using the names of the sigils it contains. Each contained sigil is itself a bounded context with its own interior language. From outside, only its name and entanglements are visible.

A context can address its siblings through their public surface. The notation `neighbor.affordance` refers to a sibling capability without crossing its boundary.

Sibling sigils can declare entanglements that describe their integration relationship. These relationships express how neighboring contexts coordinate while preserving their boundaries.
