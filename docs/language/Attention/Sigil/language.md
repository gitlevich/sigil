# Sigil

A sigil is a named pattern that attention recognizes. Its boundary makes it a bounded context. Inside that boundary, it can define up to five contained sigils.

A sigil is always contained within another sigil. What appears as a root is only the sigil currently opened within a larger context that is not in view.

From outside, a sigil is addressed by name and encountered through its public surface and entanglements. Its interior belongs to its own level of attention.
