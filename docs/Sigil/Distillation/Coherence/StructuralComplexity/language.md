# StructuralComplexity

How tangled the sigil graph is at each level of containment.

- **Breadth per level** — how many of the five slots are used, and how densely they integrate with each other.
- **Integration density** — the ratio of actual integration edges between neighbors to the maximum possible. Five neighbors with ten integrations means everything depends on everything.
- **Affordance count per sigil** — a sigil exposing many affordances is doing too much. A sigil exposing one or two is focused.
- **Depth** — how many levels before reaching leaf sigils with concrete language. Deep and wide simultaneously is a warning.
