# StructuralComplexity

How tangled the sigil graph is at each level of containment. High structural complexity makes the membrane harder to define cleanly — when everything depends on everything, boundaries blur.

- **Relationship density** — the ratio of actual relationship edges between neighbors to the maximum possible. Five neighbors with ten relationships means everything depends on everything.
- **Affordance count per sigil** — a sigil exposing many affordances is doing too much. A sigil exposing one or two is focused.
- **Depth** — how many levels before reaching leaf sigils with concrete language. Deep and wide simultaneously is a warning.
- **Breadth per level** — how many of the five slots are used at each level.
