# Navigation

Navigation is traversing the tree of trajectories. Every position in the tree is a point along a trajectory — a specific level of abstraction on a specific path from the origin.

@Ontology shows the trajectory structure explicitly: the branching paths from center to leaves, the current position within them. It is fast and precise for moving to a known destination.

@Atlas shows trajectory density spatially: the origin fills the area, contained sigils are named regions, deeper structure appears as anonymous subdivision. Dense subdivision signals high resolution — many narration points in that patch of surface. Sparse subdivision signals thin coverage. Both views reflect the same position and stay in sync.
