# UX

I use Edits to understand how a panel was shaped over time and to explore alternatives.

I can see the burst history of any panel — each burst a timestamped cluster of changes, with the upstream state it was written against. I can see what I was thinking at each stage.

When upstream changes, I can see which bursts were written before the change and haven't been confirmed since. Those are my stale decisions — things I need to revisit.

I can fork at any burst to explore a different direction. The branches accumulate as a tree of explored options, navigable and comparable.
