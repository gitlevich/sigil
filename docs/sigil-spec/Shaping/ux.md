# Shaping 

I write. I open a sigil and the UX panel is active. I narrate what the user reaches for in this context. As I write, names appear as at-references — highlighted, navigable. When I name something I Cmd+click it to enter it one level down, where I elaborate it further. Alt+Enter on an unresolved at-reference creates the sigil it names. Alt+Cmd+R renames a sigil across all references. I move between panels — UX, Language, Architecture, Implementation — as the spec deepens. I cannot contradict what I declared above.
