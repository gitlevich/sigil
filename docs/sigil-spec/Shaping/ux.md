# Shaping

I write. I open a sigil and begin narrating. I describe what the user reaches for, the concepts that make the system work, the constraints that matter, the preferences I have. As I write, names appear as at-references — highlighted, navigable. When I name something I Cmd+click it to enter it one level down, where I elaborate it further. Alt+Enter on an unresolved at-reference creates the sigil it names. Alt+Cmd+R renames a sigil across all references. Each thing I write is an edit; edits cluster into bursts. The burst history is the time-like record of how this trajectory grew.
