# Measure Name Fit

For a given @sigil, embed the sigil's name and the descriptions of all its affordances. Compute the centroid of the affordance embeddings. The distance from the name embedding to that centroid is the name fit score.

A sigil is named for the problem it solves — the affordances are what it actually provides. If the name and the affordances live in the same neighborhood of embedding space, the name is doing its job. If the name is far from the affordance centroid, it's misleading — the sigil does something other than what its name suggests.

Embedding space is @contrast space. The name should sit at the center of the contrasts its affordances span.
