# Coherence Metric

A number computed from a whole @sigil's shape in @ContrastSpace. Input: a whole sigil. Output: a number in an interpretable range. Out of range = the spec's language no longer agrees with itself.

One sigil, many projections (spec, code, visuals), one shape. If the projections disagree, the shape is fractured. The metrics detect this — drift accumulated through renames, refinements, decomposition.
