# Measure Boundary Sharpness

Subspace alignment between @invariant embeddings and content embeddings (language + affordances) in @ContrastSpace. PCA on each set, then sum of squared cosines between principal component pairs. Range: 0 (invariants describe a different subspace than the content) to 1 (invariants span exactly the subspace the content occupies). Low score = invariants are aspirational, not structural — they don't describe the actual boundary of what this @sigil is.
