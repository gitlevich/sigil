# Measure Affordance Affinity

For each affordance, the ratio of distance-to-parent vs distance-to-nearest-uncle in @ContrastSpace. Ratio < 1 = correctly placed. Ratio > 1 = closer to an uncle, probably misplaced. Aggregate: fraction of affordances with ratio < 1. Range: 0 (all misplaced) to 1 (all correctly placed).
