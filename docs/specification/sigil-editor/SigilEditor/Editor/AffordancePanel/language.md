# Affordance Panel

This is where the @sigil's affordances go. 

A @disposition is towards a thing I care about while wearing this @sigil + my preference about it. For example, I have a @disposition towards chairs along comfortable vs uncomfortable axis: I prefer them to be comfortable.

I can create a @disposition directly, via #add-disposition. Or I can have it auto-created in the @LanguagePanel, and then it will appear here, but without a description. 

I can #delete-disposition I don't want. And I can #rename-disposition and #edit-disposition-preferences
