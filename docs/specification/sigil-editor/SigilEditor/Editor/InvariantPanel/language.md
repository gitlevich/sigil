# Invariant Panel

This is where the @sigil's @invariants go.

An @invariant is what the @sigil enforces within its boundaries — a commitment expressed as a preference along a @contrast axis. For example, I have an @invariant towards chairs along comfortable vs uncomfortable: I prefer them comfortable.

I can create an @invariant directly, via #add-invariant. Or I can have it auto-created in the @LanguagePanel, and then it will appear here, but without a description.

I can #delete-invariant I don't want. And I can #rename-invariant and #edit-invariant-preferences
