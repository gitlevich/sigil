# Exploration

The self-directed movement of the @Organism that generates @Information for @Perception. Looking is not the same as seeing — looking is @Exploration. The @Organism turns its head, walks around, reaches out, manipulates. Each movement transforms the ambient array in lawful ways, revealing new @Invariants and new @Affordances. @Exploration is not random search; it is guided by what has already been perceived. The @Organism explores to resolve uncertainty and to discover what the @Environment offers.
