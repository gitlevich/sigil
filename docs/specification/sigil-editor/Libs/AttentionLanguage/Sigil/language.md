# Sigil

A sigil is a shape in embedding space where the values along the relevant contrasts it cares about are within @preferences it declares.

It can be thought of as a  a sequence of filter and map functions over @observations. 

Its main invariant is !relevant-preferences-respected.

To an @agent any sigil looks like a pattern it recognizes and about which it has a preference: less of it or more.
