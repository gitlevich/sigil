# Contrast Space

The space whose dimensions are @contrasts. Every @contrast an @observer can discriminate is an axis. A position in contrast space is a particular configuration of values along many contrasts simultaneously.

A @sigil is a shape in contrast space — a region where @preferences hold. The boundary of that shape is the sigil's @invariants. Outside the boundary, contrast values are unconstrained.

Contrast space is where structure becomes geometry. Two sigils that discriminate along similar contrasts are near each other. Two sigils that attend to entirely different contrasts are orthogonal — they don't interact. @Entanglement is intersection in contrast space.

A sufficiently rich language model's internal representation space is a contrast space: each direction is a discrimination the model has learned to make. Projecting sigil language into this space makes sigil structure — boundaries, neighborhoods, gaps — geometrically visible.
