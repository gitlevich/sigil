# Frame

An ordered set of @observations an @attention experiences at a moment.

The "moment" has some thickness: the window over which I aggregate observations on my sensors before I interpret a reading. 

Frame has a @resolution. 

Because @attention is limited, I have an upper bound on resolution. 

If @attention is sucked into a @narrative, it is limited to only a section of the frame and is probably sampling the rest at a much lower @framerate.
