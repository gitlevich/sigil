# Differential Geometry

The geometry of curved @space (@manifold) and the structures defined on them. Provides the vocabulary for distinguishing directions in a @space — which directions are @timelike, which are @spacelike, and where the @lightCone separates them.

Used by @AttentionLanguage to formalize the causal structure of @attention: @frame is @spacelike (simultaneous), @narrative is @timelike (sequential), and the @lightCone bounds what a @fixedPoint can influence at a given @resolution.
