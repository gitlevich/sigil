# Spacelike

A direction in a @manifold along which no causal signal can propagate. Events connected by a @spacelike path are simultaneous — neither influences the other. They coexist in the same moment.

The @observer can #hold-simultaneously — perceive multiple points at once. Along @spacelike directions, @observations coexist in the same @frame. The @observer can #compare — place two @observations side by side and @contrast them directly without relying on memory.

!no-causal-link: points connected by a @spacelike direction cannot influence each other. Simultaneity means independence — changing one does not change the other.

In @attention: @frame is @spacelike. All @observations in one @frame are simultaneous — the @observer holds them at once. Structure is @spacelike. Seeing the whole shape of a @sigil rather than reading about it sequentially.
