# Workspace

My @workspace consists of my entanglement surfaces. 

@Structural:

@ApplicationSpec - the @sigil in which I specify my @application. Has @ApplicationSpec#application-name affordance.
@ChatPanel - where I can chat with @DesignPartner
@MemoriesPanel - where I can spy over what @DesignPartner remembered
I use @NavigationPanel to navigate @sigils.

@Functional:

@Editor - the tool I use to work on @sigils

It maintains its !integrity offers !clear-organization of documents on disk and panels in front of me. It #remembers-its-state across @sessions.

Only one @application instance is allowed to open a @workspace. 

To maintain !integrity, many @applications can run at the same time with their own workspaces. No @applications ever share a @workspace at the same time.
