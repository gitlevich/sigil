# Private Sigil Folder

@PrivateSigilFolder is a type of @SigilFolder, where both the @user and @DesignPartner keep state not meant for publishing. 

It must !stay-local to avoid publishing private information the @user revealed to @DesignPartner.

Exposes convenience affordances #user-private, #design-partner-private and #chats.
