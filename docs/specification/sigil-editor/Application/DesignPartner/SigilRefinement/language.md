# Sigil Refinement

Sigil refinement is how I keep the @application @sigil coherent and know what holes we still have to fill: I #take-coherence-metric and #track-coverage.

Both measure the @sigil's integrity.

When coherence measured by @CoherenceMetric drops, the @sigil's shape in @AttentionLanguage@ContrastSpace deforms. It's surface is no longer smooth because narratives are contradicting each other, things get harder to express, sentences grow longer, more explanations are needed. Verbosity indicates major language suckage. It also means we can say many incoherent things in our language and it needs tightening,

When @coverage is low, the language allows correct statements that we haven't yet narrated and should. Else the coding agent will invent a narrative, inviting surprises.
