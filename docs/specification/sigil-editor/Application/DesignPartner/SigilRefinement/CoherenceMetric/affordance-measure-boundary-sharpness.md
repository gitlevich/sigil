# Measure Boundary Sharpness

Subspace alignment between @invariant embeddings and content embeddings in @ContrastSpace. High = invariants span the same subspace as the content. Low = invariants are aspirational, not structural — they don't describe the actual boundary of what this @sigil is.

An invariant that can't state its @contrast and @preference precisely is a @Coverage question, not an invariant.
