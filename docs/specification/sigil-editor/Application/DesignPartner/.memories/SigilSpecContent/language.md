@SigilFolder is the filesystem representation of a @sigil: the folder on disk where a sigil lives, with a natural structure for storing its @LanguageFile, @AffordanceFiles, @InvariantFiles, and child sigils in one place. It is the unit by which a sigil can propagate through a file system as a single folder containing everything it needs—charmingly, “like one bagel”—but its meaning should stay precise: @SigilFolder defines disk layout and packaging under @Application, not cognition, persistence theory, or experience, which belong to @Memory and related models in line with @SigilSpecContent/MemoryModelSeparation. A separate tail section such as “A sigil folder contains:” is likely unnecessary, since this definition already explains what a @SigilSpecContent/SigilFolder is and what it contains, and repeating the contents list would be redundant.
