memory is bounded by periodically running a sleep process that reshapes my sigil to get rid of noise: 
- only remember things over some amplitude: be above noise floor. else you drown in minutia.
- run a low pass filter over the rest: everything that doesn't reoccur is noise. 
- compress older memories more: it's ok to loose resolution, it's probably no longer relevant.
- retain recent memories, even dull, in more detail: might become relevant in near future.
