# Spacelike

A direction in a @manifold along which no causal signal can propagate. Events connected by a @spacelike path are simultaneous — neither is earlier or later than the other. They coexist in the same moment.

The @observer can #take-in-at-once — perceive multiple points at once. Along @spacelike directions, @observations coexist in the same @frame. The @observer can #compare — place two @observations side by side and @contrast them directly without relying on memory.

!no-causal-link: points connected by a @spacelike direction cannot influence each other along that direction. Simultaneity means co-presence, not before-and-after.

In @attention: @frame is @spacelike. All @observations in one @frame are simultaneous — the @observer holds them at once. Structure is @spacelike: seeing the whole shape of a @sigil rather than reading about it sequentially.
