# Fixed Point

The center of a @sigil's volume in @contrastSpace. The point from which @attention radiates and to which it returns. What persists when the @sigil undergoes transformation — change of @scale, @collapse, re-composition.

Every @sigil has one. It is what the name names. #measure-name-fit measures distance to it. Without a @fixedPoint, a @sigil is a list of @preferences. With one, it is an identity.

A nontrivial transformation that leaves one value unchanged. Not @attention applied to everything (that is identity). This specific @attention, stable under its own application. The @observer that observes itself and finds the same @observer.

First person: "I am that I am." The @invariant that does not need a @contrast because it is prior to discrimination. The origin from which @contrasts radiate.
