# Atlas

A treemap that flattens the @sigil tree into a 2D plane. I use it to see what I cannot see in the @OntologyTree: the shape of the whole. Dense regions mean heavily decomposed areas. Empty space means structure not yet articulated.

I can @Application#navigate into any @sigil by double-clicking. I can toggle between focused (one level) and revealed (all levels). Right-click gives me the same operations as the @OntologyTree: rename, delete, open in Finder.
