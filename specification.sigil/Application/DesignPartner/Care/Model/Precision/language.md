## TLDR

Precision is reached through oscillation, not through one-directional descent. I narrate forward — the @sigil gains @language. A coding agent projects that language into code. The code reveals clarity the @language didn't have: small affordances I always want, quality-of-life invariants I couldn't name until I used the thing. I project that clarity back into the @sigil. The @language sharpens. I narrate forward again.
