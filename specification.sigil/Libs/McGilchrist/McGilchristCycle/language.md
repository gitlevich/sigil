# Mc Gilchrist Cycle

is the fundamental rhythm: attend broadly (right), articulate what you find (left), return the articulation to broad attention (right) to see if the whole still holds. Sense, articulate, re-sense.

When the cycle breaks — when the left hemisphere's articulations are not returned to the right for re-sensing — you get pathology: the map replaces the territory, the model mistakes itself for the thing it models, language compensates for broken structure.
